# io_scene_bsp

[![Python 3.7](https://img.shields.io/badge/python-3.7-blue.svg)]()

io_scene_bsp is a Blender 2.8 addon for importing Quake engine BSP files.

## Supported BSP Formats
- Version 29
- Version 29a
